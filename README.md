Program Biaya Pengiriman

Fungsi:

Menghitung biaya total pengiriman paket dari berat, jarak, dan volume barang.

Cara kerja:

Input berat dalam kg, jarak dalam km, panjang, lebar, dan tinggi dalam cm.

Menghitung volume = panjang × lebar × tinggi.

Biaya per kg:

Jarak ≤ 10km → Rp 4250/kg

Jarak > 10 km → Rp 6000/kg

Biaya total = berat × biaya per kg.

Jika volume > 100cm³, maka tambahkan Rp 50.000.

Menampilkan hasil berupa berat, jarak, volume, biaya per kg, dan total biaya pengiriman.

Program Ganjil atau Genap

Fungsi:

Digunakan untuk menentukan apakah bilangan itu ganjil atau genap<img 

Cara kerja:

Pengguna memberikan input berupa suatu bilangan.

Program menguji apakah bilangan itu habis dibagi 2 (bilangan % 2 == 0):

o Jika benar → bilangan genap

o Jika salah → bilangan ganjil

Program mencetak hasil ke layar.
